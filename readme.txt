Benötigt:
-Node (getestet mit Version v14.17.0)
-Android Studio / installiertes + konfiguriertes JDK 

Kommandozeile im Projektordner öffnen
->"npm install" ausführen um alle nötigen Module zu installieren


->"cd android"
->"gradlew assembleRelease" um APK zu erstellen
ALTERNATIV
den Ordner "android" im Projektordner in Android Studio öffnen und darüber die APK erstellen und auf dem Gerät installieren
die APK befindet sich in "/android/app/build/outputs/apk/release"
